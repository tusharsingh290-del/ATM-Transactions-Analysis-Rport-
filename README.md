# ATM Transaction Dashboard – Power BI Project

## Overview
This project is an interactive Power BI dashboard built using an 11K+ ATM transaction dataset. The dashboard provides insights into ATM revenue, transaction trends, uptime performance, maintenance costs, and profitability across multiple Indian states.

The objective of the project was to transform raw ATM transaction data into meaningful business insights using Power BI, DAX, and data visualization techniques.

---

## Features
- Interactive KPI Dashboard
- State-wise Revenue Analysis
- Monthly Transaction Trend Analysis
- Gross Profit Percentage Tracking
- ATM Uptime Monitoring
- Cost & Maintenance Analysis
- Transaction Range Segmentation
- Dynamic Filters and Drill-down Reporting

---

## Key Insights
- Identified high-performing and low-performing states based on revenue generation.
- Analyzed ATM uptime and operational efficiency across regions.
- Compared monthly revenue and transaction trends for business monitoring.
- Evaluated maintenance costs including ATM AMC, UPS AMC, and VSAT AMC.
- Tracked gross profit percentage and operational performance metrics.
